# 🚀 Winwin 크롤러 3.3 완전 정복 가이드

환영합니다! 이 프로그램은 복잡한 도매 사이트의 상품들을 내 쇼핑몰이나 SNS(카카오스토리, 밴드)로 아주 쉽고 빠르게 옮겨주는 **업무 자동화 비서**입니다.
컴퓨터를 잘 몰라도 누구나 이 설명서의 순서만 따라 하면 마법처럼 쉽게 사용할 수 있습니다!

---

## 🏁 1. 준비 운동: 프로그램 켜고 로그인하기

프로그램이 내 대신 일을 하려면, 먼저 문을 열어주고 아이디를 알려주어야 합니다.

1. **프로그램 실행하기**
   - 바탕화면의 `Winwin 크롤러 아이콘` 혹은 폴더 안의 `실행 파일(.exe)`을 더블클릭합니다.
   - 까만 창(콘솔창)과 함께 메인 화면이 뜹니다.

2. **도매처(웨이상) 로그인하기**
   - 처음 실행하거나 로그인이 풀렸을 경우, 자동으로 크롬 인터넷 창이 하나 뜹니다.
   - 화면에 **QR 코드**가 나타나면, 스마트폰의 위챗(WeChat)이나 미상앨범 앱을 열어 QR 코드를 찍어주세요.
   - 한 번 로그인해 두면 프로그램이 스스로 기억하므로 다음번엔 묻지 않습니다!

3. **내 SNS(카카오/밴드) 로그인 정보 입력**
   - 프로그램 상단 메뉴의 ⚙️설정 아이콘을 누릅니다.
   - 내 카카오스토리와 네이버 밴드의 아이디/비밀번호를 입력해 둡니다.

---

## 🛒 2. 1단계: 도매상 물건 싹 쓸어오기 (수집하기)

이제 도매처(웨이상)에서 상품 사진과 설명을 내 컴퓨터로 긁어올 차례입니다.

1. **도매상 주소(URL) 넣기**
   - 수집하고 싶은 웨이상 업체의 앨범 인터넷 주소(URL)를 복사해서 프로그램 위쪽 빈칸에 붙여넣습니다.
   - 몇 개를 가져올지 숫자를 정해줍니다. (예: 최신 글 50개)

2. **똑똑한 번역 설정하기 (추천!)**
   - 중국어로 된 상품 설명을 한국어로 바꾸려면 **[인공지능 번역 켜기]**에 체크해 주세요.
   - **카테고리 선택**: 신발인지, 가방인지, 옷인지 선택해 주면 인공지능이 그에 맞는 예쁜 한국어 문장으로 바꿔줍니다.
   - **환율 설정**: 오늘의 네이버 환율을 적어주세요. 인공지능이 중국 돈(위안화)을 계산해서 한국 돈 판매가로 자동 변환해 줍니다.

3. **[수집 시작] 버튼 누르기**
   - 클릭 한 번이면 혼자서 알아서 물건을 하나씩 긁어옵니다. 잠시 커피 한잔하고 오세요!

---

## ✂️ 3. 2단계: 가져온 상품 입맛대로 꾸미기 (편집하기)

다 긁어왔다면, 내 쇼핑몰에 올리기 전에 예쁘게 다듬어야 합니다. 이 과정은 마우스 클릭만으로 아주 쉽게 끝납니다!

* **사진 순서 바꾸기 / 필요 없는 사진 지우기**
  - 사진을 꾹 눌러서 옆으로 끌면(드래그) 순서를 바꿀 수 있습니다.
  - 쓸데없는 사진은 각 사진에 달린 `X` 버튼을 눌러 지우세요.
  - **대표 사진 만들기**: 맨 첫 번째 칸에 놓은 사진이 무조건 메인(대표) 사진이 됩니다!

* **키보드로 빠르게 훑어보기**
  - 마우스를 안 써도 됩니다! 키보드 방향키(위 `↑`, 아래 `↓`)로 물건을 위아래로 확인하고, 올리고 싶지 않은 물건은 `스페이스바(Space)`를 눌러 체크를 해제하세요.

* **가격 마음대로 올리고 내리기**
  - "이 상품들은 마진을 1만 원 더 붙여야지!"
  - 상품들을 여러 개 선택한 뒤 가격 수정 버튼을 누르면, 원하는 금액만큼 단체로 올리거나 내릴 수 있습니다. 환율과 마진도 바로 화면에 보여서 계산기 두드릴 필요가 없습니다.

---

## 🚀 4. 3단계: 내 카카오/밴드에 자동 글 올리기 (포스팅)

예쁘게 다듬은 상품들을 이제 내 손님들이 볼 수 있게 올릴 차례입니다.

1. **올릴 물건 선택하기**
   - 화면에서 방금 수집하고 수정한 물건들 앞의 네모 박스(☑)에 체크가 되어있는지 확인하세요.

2. **어디에 올릴지 고르기**
   - 오른쪽 창에서 '카카오스토리'에 올릴지, '네이버 밴드'에 올릴지 선택합니다. (어떤 계정으로 올릴지도 고를 수 있습니다.)

3. **[업로드 시작] 누르기**
   - 버튼을 누르면 프로그램이 내 아이디로 알아서 접속해서, 사람처럼 사진을 올리고 글을 복사해서 등록합니다.
   - 인스타그램이나 카카오 정책상 로봇으로 걸려서 정지당하는 것을 막기 위해, 하나 올리고 조금 쉬었다가 다음 것을 올리는 **'자연스러운 대기 시간'**이 기본으로 설정되어 있으니 안심하세요.

4. **끝! 핸드폰 알림 받기**
   - 작업이 다 끝나면 시끄럽게 계속 울리지 않고, **"총 몇 개 올리기 성공! 몇 개 실패!"** 라고 깔끔하게 딱 한 번만 텔레그램 메신저로 알려줍니다.

## 🚨 5. 앗, 문제가 생겼어요! (자주 묻는 질문)

**Q. 프로그램이 도매상 사진을 안 가져오고 멈춰요!**
> 웨이상 로그인이 풀린 것입니다. 프로그램이 띄워주는 인터넷 창에서 스마트폰으로 다시 한번 QR 코드를 찍어주세요!

**Q. 중국 돈 가격이 한국 돈으로 계산이 안 되고 0원이라고 나와요!**
> 이 도매상이 평소와 다르게 가격을 암호처럼 꼬아서 적어두었기 때문입니다. 이럴 때는 프로그램 우측 상단 톱니바퀴(⚙️설정)에 들어가서 **"AI 업체 분석"** 버튼을 한 번 눌러주세요. 인공지능이 그 업체의 암호 패턴을 새로 공부해서 다시 정상적으로 가격을 뽑아냅니다.

**Q. 글이 엉뚱한 계정에 올라가거나 사진이 자꾸 빠져요.**
> 컴퓨터가 일하는 동안(자동 업로드 중)에는 마우스나 키보드를 마구 움직이지 않는 것이 좋습니다. 혹시라도 계속 에러가 난다면, `temp_kakao_images` 처럼 만들어진 임시 폴더를 한 번 지워주시고 다시 시작해 보세요.

**Q. 인공지능 번역이 자꾸 실패한다고 떠요.**
> 구글 제미나이(Gemini) 번역기를 사용하는 'API 키(비밀번호)'가 없거나 잘못되었을 때 나타납니다. 설정에서 API 키가 제대로 들어있는지 확인해 주세요! (또는 번역 옵션을 끄고 기본 수집으로 돌려보세요.)
